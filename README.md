# Premium Interior Design Website Showcase

A modern, responsive, and high-performance website template designed for interior design boutiques and home decor brands. This project focuses on luxury aesthetics, smooth interactions, and a "glassmorphism" design language that feels both premium and professional.

- **Glassmorphism UI**: Uses a sophisticated blend of backdrop blurs, subtle borders, and soft shadows to create a depth-rich interface.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports with a mobile-first approach.
- **Dynamic Hero Section**: Features an immersive background slider and clear call-to-actions.
- **Asymmetric Service Cards**: Custom-designed cards with unique "squaricle" rounding for a distinct brand identity.
- **Immersive Animations**: Integrated with `ScrollReveal.js` for graceful element entry and user engagement.
- **Optimized Performance**: Small footprint using vanilla CSS and JavaScript, ensuring fast load times and smooth scrolling.
- **Unified Branding**: Centralized design system using CSS variables for easy color and typography management.
